• Git : 파일 버전 관리 프로그램

  • WorkingStorage === Staging Ara === Repository === git Repository
  • Git Add : 작업 파일을 추적하라고 지시한다.
  • commit 하기전 맨 처음에 자신이 누구인지 알려준다
    git config --global user.email "you@example.com" git config --global user.name "Your Name"

  • git push -u[처음 지정시 생략할 수 있도록 하는 옵션] origin master : 서버로 올리기
